# Hono Scraper API

Production-ready social media scraping API built with Hono.js and TypeScript.

## Features

- Hono.js v4.12.27
- TypeScript strict mode
- Zod validation
- Rate limiting
- CORS support
- Structured logging with Pino
- Centralized error handling
- Consistent JSON responses
- Modular folder structure
- Multi-platform support (Bun/Node.js, Vercel, Cloudflare Workers)
